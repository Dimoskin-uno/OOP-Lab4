## Варіант завдання

* Завдання: Реалізувати класи для роботи з текстом та об’єктами морських човнів.
    * Клас `Letter` - для роботи з буквами.
    * Клас `Punctuation` - для роботи з розідловими знаками.
    * Клас `Word` – для роботи зі словами.
    * Клас `Sentence` - для роботи з реченнями.
    * Клас `Text` -  для роботи з текстом.
    * Клас `SeaBoat` – опис меблів з полями `name`, `lenght`, `speed`, `capacity`, `year`.
* Сортування: Масив човнів сортувати (довжину за зростанням) та (швидкість за спаданням).
* Пошук: Знайти ідентичний об’єкт.
